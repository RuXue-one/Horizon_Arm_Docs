装配与调试手册¶

本手册面向装配人员，帮助完成机械臂的连接、基础测试与常见问题处理。无需编程知识，按步骤执行即可。 范围说明：本手册仅覆盖当前软件提供的三项功能——连接电机、单电机控制、多电机控制。

连接对话框¶

  • 入口：连接 → 连接电机
  • “驱动板版本 X/Y”选择项：
  • 选择后，软件会自动匹配合适的回零与同步方式
  • 连接失败时，检查串口是否占用、波特率匹配、电机是否上电

版本区分¶

  • 外观识别：
  • 有屏幕的是 X 版
  • 没有屏幕的是 Y 版
  • 操作差异：Y版新增两种回零方式，可以记录断电位置回零，Y版可将指令一次性发送多个电机，详见本页“7.1 X 版 / Y 版 差异与操作指引”。

1. 使用前准备¶

  • 电脑：Windows 10/11（64 位）
  • 硬件：
  • 机械臂本体与闭环驱动板
  • USB 转 CAN 设备（如 CANable）
  • 电源与线缆（确认正负极、CAN_H、CAN_L 接线正确）
  • 软件：已安装的 Horizon_Arm 程序（桌面图标）

软件图标

安全提示：工作区域保持清空，通电后避免手靠近关节与连杆；需要停止时点击界面中的“停止/紧急停止”按钮。

2. 第一次上电与连线¶

  1. 断电状态完成接线：
  2. 电源 → 驱动板
  3. 驱动板 CAN_H/CAN_L → USB 转 CAN 设备
  4. USB 转 CAN → 电脑 USB 口
  5. 上电：
  6. 确认驱动板指示灯正常
  7. 确认机械臂各关节无明显阻滞与异常声响
  8. 电脑设备检查：
  9. 打开“设备管理器”→“端口 (COM 和 LPT)”
  10. 记住新出现的串口号（如 COM8、COM18）

在“设备管理器”中确认串口号

小贴士：常见串口号在 5~30 之间，若多个设备同时插入，请逐个拔插确认对应端口。

3. 软件连接电机¶

  1. 启动程序：双击桌面图标进入主界面。
  2. 打开连接窗口：菜单“连接 → 连接电机”。
  3. 在对话框中设置：
  4. 串口：选择上一步确认的串口（如 COM8）
  5. 波特率：500000（500K）
  6. 驱动板版本：选择 Y 或 X（影响回零模式与一次性下发选项）
  7. 电机 ID：
    • 单电机：选择“单电机 (ID=1)”
    • 双电机：选择“1,2”
    • 六关节机械臂：选择“1..6”或在“自定义ID”输入 1,2,3,4,5,6
  8. 点击“确定”开始连接：
  9. 成功：状态栏显示“已连接 X 个电机”，功能页面变为可用
  10. 失败：弹窗会列出失败的 ID 与原因
  11. 断开连接：菜单“连接 → 断开连接”（更换线缆或设备前建议先断开）

连接电机示意

4. 单电机基础测试¶

目标：验证通信正常、方向正确、关节无卡滞。

步骤： 1. 打开“电机控制 → 单电机控制”。 2. 选择电机 ID，点击“使能”。 3. 设置小角度（如 5° 或 10°）做两三次往复： - 观察运动是否平顺、无异常噪音 - 方向是否与预期一致 4. 点击“停止”，再“失能”。

提示：首次调试建议仅做 5°~10° 小角度；出现异常请点击页内“停止/紧急停止”按钮。

单电机控制演示

若方向相反或角度不对，请先不要继续整臂测试，先查找问题原因。

5. 多电机同步测试¶

目标：验证多关节协同运动与同步触发。

步骤： 1. 打开“电机控制 → 多电机控制”。 2. 在“同步控制”页：为每台电机设置小幅目标角度。 3. 启用同步选项（界面已内置同步流程）。 4. 发送后执行同步触发。 5. 观察各关节是否同时起动、到位后是否无异常。

提示：首次仅做 5°~10° 小角度；如出现异常请点击“停止/紧急停止”按钮。

多电机同步演示

提示：若个别关节未动或不同步，优先检查该关节的使能状态与连接是否稳定。

6. 常见问题与处理¶

  • 连接不上：
  • 检查串口是否选对（设备管理器确认）
  • 波特率是否为 500000，CAN 速率是否为 500K
  • 电机是否上电、ID 是否唯一
  • 线缆是否牢固，是否被其他软件占用
  • 单电机能动、多电机不同步：
  • 确认每台电机都已使能并成功读取过状态
  • 同步触发是否已执行
  • 是否已在页面中勾选/选中需要参与同步的电机
  • 角度异常、方向颠倒：
  • 在当前页面（单电机/多电机）调整减速比与方向后再次下发小角度指令进行核对
  • 运动中报错或停滞：
  • 立即点击界面“停止/紧急停止”按钮
  • 降低速度、角度，排查机械干涉与松动

6.1 X 版 / Y 版 差异与操作指引¶

  • 版型识别：

  • 外观/铭牌、固件字样或界面提示处会标注 X 或 Y（如无法判断，按下述两种流程分别试验较小目标角度，以不损伤为前提确认）。

  • 同步控制：

  • X 版：先“为每台电机下发带同步标志的目标”→ 再“发送广播同步触发”。

  • Y 版：使用“多电机一次性下发”的同步命令，无需额外广播。

  • 批量回零：

  • X 版：4 种回零模式，常用“限位回零/编码器回零”，回零后查看到位状态与零点偏差。

  • Y 版：6 种模式，包含“绝对零点/掉电位置”等；按现场需求选择，完成后执行“到位确认”。

  • 到位确认与超时：

  • X 版：按常规定时/返回包确认。

  • Y 版：运动类默认仅地址 1 确认且不等待，以避免超时；需要严格等待时，开启“等待到位/超时”选项并设置合理超时（建议 3~8 秒，小角度更短）。

  • 修改电机 ID：

  • 两版均支持；Y 版支持 AE 4B 指令，可选择“仅临时/写入存储”。批量改 ID 前，先断开其他设备，逐台进行，避免冲突。

小贴士：两版混用时，优先按 Y 版“多电机一次性下发”方案测试；若个别关节不响应，再切回 X 版“同步标志+广播触发”流程验证。

点击界面上的“停止/紧急停止”按钮

7. 术语简明解释¶

  • 串口：电脑上的通信编号（如 COM8），选择对了才能通信
  • 波特率：通信速度，本系统固定为 500000（500K）
  • 电机 ID：给每台电机分配的编号，避免冲突才能同时控制
  • 同步：多台电机同时动作的机制

8. 安全与记录¶

  • 紧急停止：在界面中使用“停止/紧急停止”按钮
  • 现场记录：
  • 拍照或录像保存异常现象
  • 记录串口号、连接的电机 ID、发生时间
  • 便于技术支持定位问题

更多界面说明与进阶功能，请参阅“用户指南”。
